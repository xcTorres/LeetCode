#### Remove Element  
problem: [https://leetcode.com/problems/remove-element/](https://leetcode.com/problems/remove-element/)  

```java

    class Solution {
        public int removeElement(int[] nums, int val) {
            
            int idx = 0;
            for(int i=0; i<nums.length; i++){
                
                if( nums[i] == val ){
                    continue;        
                }else{
                    nums[idx++] = nums[i];    
                }
            }
            return idx;
        }
    }

```
